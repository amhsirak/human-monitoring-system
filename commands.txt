1. To run inference on a test video file
    For people example:
    python main.py --prototxt mobilenet_ssd/MobileNetSSD_deploy.prototxt --model mobilenet_ssd/MobileNetSSD_deploy.caffemodel --input videos/people.mp4
    
    For night example:
    python run.py --prototxt mobilenet_ssd/MobileNetSSD_deploy.prototxt --model mobilenet_ssd/MobileNetSSD_deploy.caffemodel --input videos/night.mp4


2. To run inference on an IP camera or webcam:
    Setup your camera url in 'mylib/config.py':
    # Enter the ip camera url (e.g., url = 'http://191.138.0.100:8040/video')
    url = ''
    python main.py --prototxt mobilenet_ssd/MobileNetSSD_deploy.prototxt --model mobilenet_ssd/MobileNetSSD_deploy.caffemodel
    Set url = 0 for webcam.
